【作品名】Biscuit&Strap
【概要】ジャイロセンサを用いた2D横スクロールシューティング&ランゲーム
【動作環境】PC/iOS/Android
【開発環境】Unity/C#
【開発人数】3人
【担当箇所】企画/プログラム/デザイン
【製作期間】約2か月

-具体的には、以下のプログラムを担当しました。全ソースコードは Assets/Scripts から閲覧できます。
--タイトル画面
--メニュー画面
--インベントリ機能
--ストア(銃のガチャ)
--オプションの各種設定
--セーブデータの管理
--物体の基本的挙動

--各エフェクトの作成と発生
